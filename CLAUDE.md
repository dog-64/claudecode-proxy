# CloudMD — правила проекта claudecode-proxy

## Общие правила

- После **каждых правок** скриптов `.sh` или bash-скриптов обязательно запускать `shellcheck` для проверки:
  ```bash
  shellcheck <файл>
  ```
  Исправлять все найденные ошибки и предупреждения перед коммитом.

## Специфика проекта

### Скрипт clc

- **GLM модель**: работает напрямую, БЕЗ прокси
  - API Z.ai доступен из России
  - Отключается прокси через `NO_PROXY="*"` и `unset http_proxy/https_proxy`
  - Туннель не запускается

- **Sonnet/Opus/Haiku модели**: работают через прокси
  - API Anthropic заблокирован в России
  - Запускается SSH-туннель к dog@vdsina
  - Устанавливаются переменные `http_proxy` и `https_proxy`
