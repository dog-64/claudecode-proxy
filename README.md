## 🚀 Быстрая установка

Просто запусти скрипт установки:
```bash
./install.sh
```

Скрипт:
- 🔍 Найдёт все bin-каталоги в домашней папке
- 📂 Предложит выбрать каталог или создать новый
- 📋 Скопирует файлы и сделает их исполняемыми
- 🔍 Проверит PATH и покажет инструкции для настройки shell

## 🛠️ Ручная установка

### 1. Убедись, что каталог существует:
```bash
mkdir -p ~/.local/bin
```

### 2. Скопируй скрипт:
```bash
cp clc ~/.local/bin/
```

### 3. Сделай исполняемым:
```bash
chmod +x ~/.local/bin/clc
```
