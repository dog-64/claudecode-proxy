# Claude Code Proxy

Скрипт `clc` для запуска Claude Code через SSH-туннель и squid-прокси с поддержкой выбора модели (GLM или Sonnet).

## 🚀 Быстрая установка

Просто запусти скрипт установки:
```bash
./install.sh
```

Скрипт:
- 🔍 Найдёт все bin-каталоги в домашней папке
- 📂 Предложит выбрать каталог или создать новый
- 📋 Скопирует файлы и сделает их исполняемыми
- 🔍 Проверит PATH и покажет инструкции для настройки shell

## 🛠️ Ручная установка

### 1. Убедись, что каталог существует:
```bash
mkdir -p ~/.local/bin
```

### 2. Скопируй скрипт:
```bash
cp clc ~/.local/bin/
```

### 3. Сделай исполняемым:
```bash
chmod +x ~/.local/bin/clc
```

## ⚙️ Настройка переменных окружения

### 1. Токен Z.ai (для моделей GLM)

```bash
# Добавь в ~/.zshrc или ~/.bashrc
export ZAI_API_TOKEN="твой_токен_zai"
```

**Где взять токен:**
- Зайди на https://docs.z.ai/devpack/tool/claude
- Получи API ключ для GLM Coding Plan

### 2. Настройка прокси (для моделей Sonnet/Opus/Haiku)

Для работы с моделями Anthropic необходим SSH-туннель к прокси-серверу:

```bash
# Добавь в ~/.zshrc или ~/.bashrc
export PROXY_USER="твой_ssh_пользователь"
export PROXY_HOST="твой_прокси_хост"
```

**Пример:**
```bash
export PROXY_USER="user"
export PROXY_HOST="vdsina"
```

**Требования к прокси-серверу:**
- SSH-доступ по ключу (без пароля)
- Запущенный squid на порту 3128
- Настройка в `~/.ssh/config` (опционально)

Если переменные не заданы, скрипт использует значения по умолчанию и покажет предупреждение.

## 📖 Использование

### Базовый запуск (по умолчанию GLM):
```bash
clc
```

### Явный выбор модели:
```bash
# GLM модель (Z.ai)
clc --model glm

# Sonnet модель (оригинальный Anthropic)
clc --model sonnet
```

### Передача других аргументов Claude:
```bash
clc --model sonnet /help
clc --model glm /status
```

### Особые команды (без прокси):
```bash
clc update    # Обновление Claude Code
clc mcp       # Работа с MCP
```

**Что происходит при выборе модели:**

- **GLM**: Перенаправляет запросы моделей на Z.ai API
  - Работает **напрямую, БЕЗ прокси** (API Z.ai доступен из России)
  - `ANTHROPIC_AUTH_TOKEN` — берётся из `ZAI_API_TOKEN`
  - `ANTHROPIC_BASE_URL` — `https://api.z.ai/api/anthropic`
  - `ANTHROPIC_DEFAULT_HAIKU_MODEL` → `glm-4.5-air`
  - `ANTHROPIC_DEFAULT_SONNET_MODEL` → `glm-4.7`
  - `ANTHROPIC_DEFAULT_OPUS_MODEL` → `glm-4.7`

  Когда Claude запрашивает Sonnet → фактически используется GLM-4.7

- **Sonnet/Opus/Haiku**: Используется оригинальный API Anthropic со стандартными моделями
  - Работает **через прокси** (API Anthropic заблокирован в России)

## ✅ Проверка: всё ли работает

